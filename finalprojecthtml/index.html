Kaggle Competition: House Prices



We joined the Kaggle competition to predict house prices in Ames, Iowa based on 79 variables that describe various aspects of a home.

We downloaded the data which was split into a training set and a testing set to use to make final predictions.

We then explored the data which helped us to clean it and make it ready for modeling.

Next we began modeling and evaluated the performance and visualized the results in an attempt to improve our results.

We then made predictions on inputs we made up using select features.

Step by Step Process

Understand the Problem - Look at each variable and do a philosophical analysis about its meaning and importance for this problem.

Univariable Study - Focus on the dependent variable (“SalePrice”) and try to know a little bit more about it.

Multivariate Study - Try to understand how the dependent variable and independent variables relate.

Basic Cleaning - Clean the dataset and handle the missing data, outliers and categorical variables

Test Assumptions - Check if our data meets the assumptions required by most multivariate techniques.
